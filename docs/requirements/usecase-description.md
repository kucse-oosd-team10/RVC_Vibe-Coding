# RVC Use Case Description

## 1. 현재 단계

현재 단계는 Requirements Analysis이다. 본 문서는 `rvc-controller`의 자동 진공 청소 제어 로직 유스케이스를 정의한다.

## 2. 액터

| 액터 | 설명 |
|---|---|
| 사용자 | 청소 시작 및 종료 버튼을 누르는 외부 액터 |
| 전방 센서 하드웨어 | 전방 장애물 interrupt 입력을 제공하는 외부 actor |
| 측면 센서 하드웨어 | 좌측 및 우측 장애물 polling 입력을 제공하는 외부 actor |
| 먼지 센서 하드웨어 | 먼지 감지 입력을 제공하는 외부 actor |
| Movement Motor | `rvc-controller`의 이동 명령을 수신하는 외부 actuator |
| Cleaning Motor | `rvc-controller`의 흡입 명령을 수신하는 외부 actuator |

## 3. 유스케이스 목록

| ID | 유스케이스 | 주요 관련 요구사항 |
|---|---|---|
| UC-001 | 자동 진공 청소 시작 | RVC-FR-001, RVC-FR-021, RVC-FR-022, RVC-FR-030, RVC-FR-031, RVC-FR-032 |
| UC-002 | 자동 진공 청소 종료 | RVC-FR-002, RVC-FR-020, RVC-FR-030, RVC-FR-031, RVC-FR-033 |
| UC-003 | 전방 장애물 회피 | RVC-FR-004 ~ RVC-FR-010, RVC-FR-015, RVC-FR-016, RVC-FR-030 |
| UC-004 | 삼방향 장애물 탈출 | RVC-FR-011 ~ RVC-FR-016, RVC-FR-030 |
| UC-005 | 먼지 감지에 따른 흡입 강화 | RVC-FR-017 ~ RVC-FR-029, RVC-FR-031 |

## 4. UC-001 자동 진공 청소 시작

| 항목 | 내용 |
|---|---|
| 주요 액터 | 사용자 |
| 목적 | 사용자가 버튼 입력으로 RVC의 자동 진공 청소를 시작한다. |
| 사전조건 | RVC가 OFF 상태이다. |
| 사후조건 | RVC는 전원 ON 상태가 되고 NORMAL 흡입 상태로 전진을 시작한다. |
| 관련 요구사항 | RVC-FR-001, RVC-FR-021, RVC-FR-022, RVC-FR-003, RVC-FR-030, RVC-FR-031, RVC-FR-032 |

### 기본 흐름

1. 사용자가 청소 시작 버튼을 입력한다.
2. RVC는 OFF 상태에서 전원 ON 상태로 전환한다.
3. RVC는 전방 센서 하드웨어, 측면 센서 하드웨어, 먼지 센서 하드웨어와의 센서 인터페이스를 초기화한다.
4. RVC는 cleaning motor에 NORMAL 흡입 명령을 제공한다.
5. RVC는 movement motor에 전진 명령을 제공한다.

### 예외 흐름

- 현재 요구사항에서는 별도 예외 흐름을 정의하지 않는다.

## 5. UC-002 자동 진공 청소 종료

| 항목 | 내용 |
|---|---|
| 주요 액터 | 사용자 |
| 목적 | 사용자가 버튼 입력으로 RVC의 자동 진공 청소를 종료한다. |
| 사전조건 | RVC가 전원 ON 상태이다. |
| 사후조건 | RVC는 OFF 상태가 되고 자동 청소 제어 로직과 진공 흡입 동작을 수행하지 않는다. |
| 관련 요구사항 | RVC-FR-002, RVC-FR-020, RVC-FR-030, RVC-FR-031, RVC-FR-033 |

### 기본 흐름

1. 사용자가 청소 종료 버튼을 입력한다.
2. RVC는 청소 동작을 종료한다.
3. RVC는 movement motor에 정지 명령을 제공한다.
4. RVC는 cleaning motor에 OFF 흡입 명령을 제공한다.
5. RVC는 전방 센서 하드웨어, 측면 센서 하드웨어, 먼지 센서 하드웨어와의 센서 인터페이스를 종료한다.
6. RVC는 OFF 상태로 전환한다.

### 예외 흐름

- 청소 종료 버튼 입력이 회전, 후진, POWER_UP 중 발생했을 때의 처리 우선순위는 아직 확정되지 않았다.

## 6. UC-003 전방 장애물 회피

| 항목 | 내용 |
|---|---|
| 주요 액터 | 전방 센서 하드웨어, 측면 센서 하드웨어, Movement Motor |
| 목적 | 전진 중 전방 장애물이 감지되었을 때 좌우 장애물 상태에 따라 회피 방향을 결정한다. |
| 사전조건 | RVC가 전진 중이다. |
| 사후조건 | RVC는 회전 완료 후 다시 전진한다. |
| 관련 요구사항 | RVC-FR-004 ~ RVC-FR-010, RVC-FR-015, RVC-FR-016, RVC-FR-030 |

### 기본 흐름

1. 전방 센서 하드웨어가 전방 장애물 interrupt를 제공한다.
2. RVC는 movement motor에 정지 명령을 제공한다.
3. RVC는 측면 센서 하드웨어에서 좌측 및 우측 장애물 값을 확인한다.
4. 전방 장애물만 감지된 경우 RVC는 movement motor에 좌회전 명령을 제공한다.
5. 전방 및 좌측 장애물이 감지된 경우 RVC는 movement motor에 우회전 명령을 제공한다.
6. 전방 및 우측 장애물이 감지된 경우 RVC는 movement motor에 좌회전 명령을 제공한다.
7. 회전 모드 진입 후 10초가 지나면 RVC는 회전 완료로 간주한다.
8. RVC는 다시 전진한다.

### 대안 흐름

- 전방, 좌측, 우측 장애물이 모두 감지되면 UC-004 삼방향 장애물 탈출로 이어진다.

## 7. UC-004 삼방향 장애물 탈출

| 항목 | 내용 |
|---|---|
| 주요 액터 | 전방 센서 하드웨어, 측면 센서 하드웨어, Movement Motor |
| 목적 | 전방, 좌측, 우측 장애물이 모두 감지된 상황에서 후진 후 장애물이 해제된 방향으로 탈출한다. |
| 사전조건 | 전방, 좌측, 우측 장애물이 모두 감지되었다. |
| 사후조건 | RVC는 회전 완료 후 다시 전진한다. |
| 관련 요구사항 | RVC-FR-011 ~ RVC-FR-016, RVC-FR-030 |

### 기본 흐름

1. RVC는 전방, 좌측, 우측 장애물이 모두 감지된 것을 확인한다.
2. RVC는 movement motor에 정지 명령을 제공한다.
3. RVC는 movement motor에 후진 명령을 제공한다.
4. RVC는 후진 중 측면 센서 하드웨어에서 좌측 및 우측 장애물 값을 확인한다.
5. 좌측 장애물이 먼저 감지되지 않게 되면 RVC는 movement motor에 좌회전 명령을 제공한다.
6. 우측 장애물이 먼저 감지되지 않게 되면 RVC는 movement motor에 우회전 명령을 제공한다.
7. 좌측 및 우측 장애물이 동시에 감지되지 않게 되면 RVC는 movement motor에 좌회전 명령을 제공한다.
8. 회전 모드 진입 후 10초가 지나면 RVC는 회전 완료로 간주한다.
9. RVC는 다시 전진한다.

### 예외 흐름

- 후방 장애물은 현재 요구사항 범위에서 감지하지 않는다.
- 후진 최대 허용 시간은 현재 요구사항 범위에서 정의하지 않는다.

## 8. UC-005 먼지 감지에 따른 흡입 강화

| 항목 | 내용 |
|---|---|
| 주요 액터 | 먼지 센서 하드웨어, Cleaning Motor |
| 목적 | 먼지가 감지되면 진공 흡입 상태를 POWER_UP으로 전환하고 3초 단위로 유지 여부를 판단한다. |
| 사전조건 | RVC가 전원 ON 상태이다. |
| 사후조건 | 먼지가 계속 감지되면 POWER_UP이 유지되고, 먼지가 감지되지 않으면 NORMAL로 복귀한다. |
| 관련 요구사항 | RVC-FR-017 ~ RVC-FR-029, RVC-FR-031 |

### 기본 흐름

1. 먼지 센서 하드웨어가 먼지 감지 입력을 제공한다.
2. RVC는 cleaning motor에 POWER_UP 흡입 명령을 제공한다.
3. RVC는 POWER_UP 상태를 3초 동안 유지한다.
4. 3초 경과 후 RVC는 먼지 감지 값을 다시 확인한다.
5. 먼지가 다시 감지되면 RVC는 POWER_UP 상태를 유지하고 3초 타이머를 다시 시작한다.
6. 먼지가 감지되지 않으면 RVC는 cleaning motor에 NORMAL 흡입 명령을 제공한다.

### 대안 흐름

1. 정지, 회전, 후진 중 먼지가 감지된다.
2. RVC는 즉시 POWER_UP으로 전환하지 않고 NORMAL 상태를 유지한다.
3. 이후 전진을 시작할 때 POWER_UP으로 진입한다.

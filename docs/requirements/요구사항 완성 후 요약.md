Preliminary requirements 이후에 사용자가 추가로 정의한 사항은 다음입니다.

  기능 범위

  - 자동 청소 대상은 진공 청소 제어 로직으로 한정한다.
  - 기존 preliminary requirements에 있던 청소 외 부가 기능은 구현 범위에서 제외한다.
  - 실제 하드웨어 구현은 제외하고, rvc-controller의 제어 로직과 외부 인터페이스를 중심으로 정의한다.

  시스템 바운더리

  - 개발 대상 시스템은 rvc-controller이다.
  - 실제 센서 하드웨어와 actuator는 rvc-controller 바운더리 밖의 외부 actor로 정의한다.
  - 외부 actor는 다음과 같다.
      - 사용자
      - 전방 센서 하드웨어
      - 측면 센서 하드웨어
      - 먼지 센서 하드웨어
      - Movement Motor
      - Cleaning Motor

  센서 입력

  - 장애물은 전방, 좌측, 우측 세 방향의 센서 값으로 판단한다.
  - 전방 장애물 센서는 interrupt 방식이다.
  - 좌측/우측 측면 센서는 polling 방식이다.
  - 전방 장애물이 없으면 좌우 센서 값은 회피 판단에 사용하지 않는다.
  - 전방 센서를 회피 판단의 우선 입력으로 사용한다.
  - 후진 중 후방 장애물은 감지할 수 없으며, 현재 요구사항에서는 후방 장애물이 발생하지 않는다고 가정한다.
  - 후진 중 좌우 센서 polling 주기는 나중에 정의한다.
  - 청소 시작 시 센서 인터페이스를 초기화한다.
  - 청소 종료 시 센서 인터페이스를 종료한다.

  이동/장애물 회피 규칙

  - 좌회전을 기본 우선 방향으로 한다.
  - 전방 장애물만 감지되면 정지 후 좌회전한다.
  - 전방+좌측 장애물이 감지되면 정지 후 우회전한다.
  - 전방+우측 장애물이 감지되면 정지 후 좌회전한다.
  - 전방+좌측+우측 장애물이 모두 감지되면 정지 후 후진한다.
  - 후진 중 좌우 센서를 확인하다가 장애물이 없어진 쪽으로 회전한다.
  - 후진 중 좌우 장애물이 동시에 사라지면 좌회전한다.
  - 회전 모드에 진입한 뒤 10초가 지나면 회전이 성공했다고 가정한다.
  - 후진 최대 허용 시간은 현재 요구사항에서는 고려하지 않는다.
  - 후진 상태가 오래 지속될 때의 예외 처리나 360도 회전 기능은 추후 확장 후보로 둔다.

  흡입 상태

  - 진공 흡입 상태는 OFF, NORMAL, POWER_UP 3단계이다.
  - OFF는 전체 시스템 전원 OFF를 의미한다.
  - 청소 시작 시 OFF → NORMAL로 전환한다.
  - 전원이 켜져 있는 동안 진공 모터는 NORMAL 또는 POWER_UP 상태를 가진다.
  - 먼지가 감지되지 않는 일반 상황에서는 NORMAL을 유지한다.
  - 전진 중 먼지가 감지되면 POWER_UP으로 전환한다.
  - POWER_UP은 3초 동안 유지한다.
  - 3초 후 먼지 값을 다시 읽는다.
  - 3초 후 먼지가 다시 감지되면 NORMAL로 돌아가지 않고 POWER_UP 3초 타이머를 재시작한다.
  - 3초 후 먼지가 감지되지 않으면 NORMAL로 복귀한다.
  - 정지, 회전, 후진 중에는 흡입 상태를 NORMAL로 유지한다.
  - 정지, 회전, 후진 중 먼지가 감지되면 즉시 POWER_UP으로 전환하지 않고, 이후 전진을 시작할 때 POWER_UP으로 진입한다.

  사용자 입력

  - 청소 시작은 사용자 버튼 입력으로 trigger된다.
  - 청소 종료도 사용자 버튼 입력으로 trigger된다.
  - 청소 종료 버튼 입력이 회전, 후진, POWER_UP 중 발생했을 때의 처리 우선순위는 아직 미정이다.

  외부 actuator 인터페이스

  - rvc-controller는 Movement Motor에 추상 이동 명령을 제공한다.
      - 전진
      - 정지
      - 후진
      - 좌회전
      - 우회전
  - rvc-controller는 Cleaning Motor에 추상 흡입 명령을 제공한다.
      - OFF
      - NORMAL
      - POWER_UP
